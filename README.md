# MyHealth monorepo

✨ All public libraries for MyHealth frontend applications ✨.  
The monorepo is managed as an [Nx Workspace](https://nx.dev/).

## Packages

- [@myhealth-belgium/vitals](./packages/vitals/components/README.md) — Lit web component library
- [@vitals/core](./packages/vitals/core/) — Shared types, i18n, and utilities
- [@vitals/tokens](./packages/vitals/tokens/) — CSS design tokens
